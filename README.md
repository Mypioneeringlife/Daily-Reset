# The Daily Reset PWA

A publish-ready Progressive Web App version of The Daily Reset.

## Files

- `index.html` — complete app
- `manifest.json` — installable app metadata
- `service-worker.js` — offline cache support
- `icons/` — app icons
- `.nojekyll` — GitHub Pages helper

## Publish on GitHub Pages

1. Create a GitHub repository.
2. Upload all files in this folder to the root of the repository.
3. Go to **Settings → Pages**.
4. Set source to **Deploy from a branch**.
5. Choose branch **main** and folder **/root**.
6. Open the GitHub Pages URL on your phone.

## Install on Android

1. Open the GitHub Pages URL in Chrome.
2. Tap the three dots.
3. Tap **Add to Home screen** or **Install app**.
4. The Daily Reset will appear on your phone like an app.

## Important

The app stores your data locally in your browser. Use the JSON backup feature before clearing browser data or changing phones.
